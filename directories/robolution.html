Robolution 2024: Coding Challenges

By Erick Gaceta • 5 min read

Participants at the Robolution 2024 event

The Robolution event organized by Erovoutika at the University of Baguio revealed significant challenges in our Arduino programming approach, resulting in non-functional robots throughout the competition.

Day 1: Orientation

The event began with an orientation session where we learned about the competition structure and rules. This day was dedicated solely to understanding the event format with no technical inspections or testing.

Day 2: Line Following Challenges

The line following events exposed our Arduino coding issues as the robots failed to move. Problems included incorrect loop structures and improper sensor value handling in our sketches.

Day 3: Sumobot Competition

Our rewritten code still couldn't execute properly during the Sumobot matches. The afternoon featured a robotics quiz where we demonstrated our theoretical knowledge of Arduino programming.

Programming Lessons

This experience taught us valuable lessons about proper code structure, debugging techniques, and the importance of incremental testing in robotics projects. We've since focused on improving our Arduino programming skills through structured practice.
